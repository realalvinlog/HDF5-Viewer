## 🎉 发布说明

HDF5 Viewer v0.1.0 是一个轻量级的 HDF5 文件查看器，采用 VSCode 风格界面设计。

### 核心特性

- **多标签页 + Split** — 支持多文件同时查看，支持分屏
- **大数据懒加载** — 异步加载，UI 不卡死
- **智能切片** — 默认限制 200x100，避免大数据卡顿
- **数据导出** — 支持导出为 CSV 格式
- **插件系统** — 可扩展的分析和可视化插件
- **VSCode 风格** — 熟悉的界面设计

## 📦 打包信息

### Linux 版本

- **完整版**: `dist/HDF5Viewer/` (216MB)
- **便携版**: `dist/HDF5Viewer-0.1.0-linux-portable/` (216MB)

### 启动方式

```bash
# 完整版
./dist/HDF5Viewer/run.sh your_file.h5

# 便携版
./dist/HDF5Viewer-0.1.0-linux-portable/run.sh your_file.h5
```

### Windows 版本

需要在 Windows 系统上构建：

```bash
# 安装依赖
pip install PyQt6 h5py numpy pyinstaller

# 构建
pyinstaller hdf5viewer_windows.spec --noconfirm
```
